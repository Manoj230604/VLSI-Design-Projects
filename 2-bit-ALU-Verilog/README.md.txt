# 2-bit ALU Design in Verilog

This project implements a 2-bit Arithmetic Logic Unit (ALU) using Verilog HDL.

## Features
- Addition
- Subtraction
- AND
- OR
- XOR

## Tools Used
- Verilog HDL
- EDA Playground
- EPWave waveform viewer

## Inputs
A[1:0] – Operand A  
B[1:0] – Operand B  
Sel[2:0] – Operation selector  

## Output
Result[2:0] – ALU Output
